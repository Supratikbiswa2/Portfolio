# **App Name**: FaceAttend

## Core Features:

- User Details Access: Admin: View details of all students and faculties.
- Timetable Management: Admin: Create and manage timetables.
- Attendance Marking: Faculty: Mark student attendance (absent/present) for classes.
- Face ID Authentication: Student: Mark attendance by showing Face ID (via camera integration).
- Attendance Reporting: Student: View personalized attendance reports with total classes, attended classes, and attendance percentage.
- Attendance Alerts: Student: Receive alerts for low attendance and notifications of missed classes.
- Attendance Constraint Validation: Use AI to determine if constraints such as the location and time match the students class before logging attendance; Use a tool to add information on failed attempts and classify them.

## Style Guidelines:

- Primary color: A calming, professional blue (#29ABE2) to evoke trust and reliability.
- Background color: A light, desaturated blue (#E5F6FD) to provide a clean and unobtrusive backdrop.
- Accent color: A vibrant green (#90EE90) to highlight successful actions and important information.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined look, providing clear readability for reports and timetables.
- Use simple, geometric icons to represent attendance status, user roles, and alert types.
- Employ a clean, card-based layout for displaying student information and attendance records.
- Use subtle animations to acknowledge attendance submission and display alerts.
